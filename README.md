# AAPOC — Site Institucional

Site institucional da **AAPOC MT — Associação de Apoio aos Pacientes Oncológicos de Cuiabá**, desenvolvido para apresentar a associação, seus projetos, serviços e canais de contato ao público.

## Sobre a AAPOC

A AAPOC foi fundada em dezembro de 2020 por Janaína, ela própria uma paciente oncológica, com a missão de levar **dignidade, amor e esperança** a pacientes que enfrentam o câncer em Cuiabá — MT. A associação oferece assistência integral a homens, mulheres e crianças em tratamento oncológico.

## Seções do Site

| Seção | Descrição |
|---|---|
| **Início** | Apresentação da missão e números da associação |
| **Calendário** | Agenda de eventos e atividades |
| **Serviços** | Equipe multidisciplinar de voluntários (psicologia, fisioterapia, odontologia, jurídico) |
| **Nossa História** | Origem e trajetória da AAPOC |
| **Projetos** | Os 10 projetos ativos, cada um com modal de detalhes |
| **Galeria** | Fotos das ações e eventos |
| **Como Ajudar** | Formas de contribuir com a associação |
| **Contato** | Endereço, WhatsApp, Instagram e mapa |

## Stack Técnica

- **React 18** + **TypeScript**
- **Vite** — bundler e dev server
- **Tailwind CSS** — estilização
- **shadcn/ui** + **Radix UI** — componentes de interface
- **Embla Carousel** — carrossel de profissionais
- **React Hook Form** + **Zod** — formulário de voluntariado
- **Vitest** + **Testing Library** — testes

## Como Rodar

```bash
# Instalar dependências
npm install

# Servidor de desenvolvimento
npm run dev

# Build de produção
npm run build

# Preview do build
npm run preview

# Testes
npm test
```
